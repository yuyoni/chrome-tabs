# ⌨️ Smart Tab Manager - 단축키 가이드

## 기본 단축키

### 🔍 탭 검색 열기

- **모든 OS**: `Alt + S` (Search)

팝업 창이 열리고 바로 검색창에 포커스됩니다.

---

### 📁 수동 탭 그룹화

- **모든 OS**: `Alt + G` (Group)

15개 미만의 탭이 있어도 즉시 그룹화를 실행합니다.

---

## 단축키 사용 시나리오

### 시나리오 1: 빠른 탭 찾기

1. `Alt + S`
2. 키워드 입력 (예: "react", "github")
3. 검색 결과에서 "Open" 클릭

### 시나리오 2: 탭 정리하기

1. 탭이 많아서 정리가 필요할 때
2. `Alt + G`
3. 자동으로 도메인별로 그룹화됨

---

## 단축키 변경 방법

Chrome에서 단축키를 커스터마이즈할 수 있습니다:

1. Chrome 열기
2. 주소창에 입력: `chrome://extensions/shortcuts`
3. "Smart Tab Manager" 찾기
4. 원하는 단축키로 변경

### 추천 대체 단축키

| 기능    | 기본 단축키 | 대체 추천                             |
| ------- | ----------- | ------------------------------------- |
| 탭 검색 | `Alt + S`   | `Alt + T`, `Ctrl + Shift + Y`         |
| 그룹화  | `Alt + G`   | `Alt + Shift + G`, `Ctrl + Shift + X` |

## 팁 & 트릭

### 💡 빠른 워크플로우

```
1. 여러 탭 열기
2. Alt + G → 즉시 그룹화
3. 작업 완료 후 탭 닫기
4. 나중에 Alt + S → 검색해서 다시 열기
```

### 💡 검색 후 바로 이동

- 팝업 열림 → 검색창 자동 포커스
- 키워드 입력
- `Tab` 키로 결과 이동
- `Enter`로 탭 열기

---

## 단축키 충돌 해결

다른 확장 프로그램과 충돌하는 경우:

### 일반적인 충돌

- `Alt + S`: 일부 확장과 충돌 가능
    - **해결**: `chrome://extensions/shortcuts`에서 `Alt + T` 또는 `Ctrl + Shift + Y`로 변경

- `Alt + G`: Google 번역 확장과 충돌 가능
    - **해결**: `chrome://extensions/shortcuts`에서 `Alt + Shift + G`로 변경

### ✅ 충돌 없는 단축키 (이미 변경됨)

기본 Chrome 단축키(`Ctrl+Shift+F`, `Ctrl+Shift+G`)와의 충돌을 피하기 위해
`Alt + S` / `Alt + G`를 기본값으로 설정했습니다.

---

## 키보드만으로 사용하기

### 전체 플로우 (마우스 없이)

1. **팝업 열기**: `Alt + S`
2. **검색어 입력**: 자동으로 포커스됨
3. **결과 탐색**: `Tab` 또는 `↓` / `↑`
4. **탭 열기**: `Enter` (결과에 포커스된 상태)

### 그룹화 플로우

1. **수동 그룹화**: `Alt + G`
2. **완료**: 알림 없이 백그라운드에서 실행

---

## 단축키 비활성화

단축키를 사용하지 않으려면:

1. `chrome://extensions/shortcuts` 접속
2. "Smart Tab Manager" 찾기
3. 각 단축키 옆 "×" 클릭하여 제거

---

## 문제 해결

### Q: 단축키가 작동하지 않아요

**A**:

- Chrome이 포커스되어 있는지 확인
- `chrome://extensions/shortcuts`에서 설정 확인
- 다른 확장과 충돌 확인
- Chrome 재시작

### Q: 단축키를 눌렀는데 아무 일도 안 일어나요

**A**:

- Chrome 개발자 도구 열기 (F12)
- Console 탭에서 에러 확인
- 확장 프로그램 재설치

### Q: Mac에서도 같은 단축키인가요?

**A**:

- 네! Mac에서도 `Alt + S`, `Alt + G` 동일
- Mac의 Alt 키 = Option (⌥) 키
